Opening Files in PRIDE Inspector

Previous | Next | See Also

This chapter explains how to view files with PRIDE Inspector.
You can open files in 3 different ways:

  • From the Welcome screen, clicking on the Open Identification of Peak Files icon
  • From the File | Open File option in the menu bar
  • From the toolbar, clicking on the button
You can open multiple files at the same time (same or different format) by clicking Ctrl and selecting with the mouse the different files you want to inspect.
mIdentML and mzTab files do not contain the spectra information inside the result file, this information is referenced from the file. If the user wants to load the spectra information related with the identifications, it need to add the spectra files (Read More)
