What is PRIDE XML Format?

Previous | Next | See Also

The PRIDE XML is created to store both spectrum and protein identification related information, in addition, it allows experiment metadata to captured. PRIDE XML files contains the identification results and the spectra data in the same file. Currently, It is one of the format to use if you would like to submit your data to PRIDE. PRIDE XML files should be generated using PRIDE Converter tool to be fully compatible with PRIDE submission process.
