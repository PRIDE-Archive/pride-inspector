What is PRIDE?

Previous | Next | See Also

The PRIDE database provides a publicly accessible, centralized data repository for mass spectrometry (MS) based proteomics data. PRIDE stores protein and peptide identifications, spectra and valuable metadata. PRIDE also allows data to remain private during peer review, while allowing optional access to collaborators, journal editors, and peer reviewers. Several journals recommend PRIDE as platform for data sharing and dissemination.
